2. **Create a New Post**
   - Go to `/admin/posts/new`
   - Enter title (auto-generates slug)
   - Write content using Markdown
   - Select category (optional)
   - Select tags (multiple)
   - Add excerpt (optional)
   - Select status (draft or published)
   - Configure SEO (optional)
   - Click "Save Draft" or "Publish"

### Editing Content

1. **Edit a Page**
   - Go to Pages list
   - Click edit on a page
   - Make changes
   - Save draft or publish

2. **Edit a Post**
   - Go to Posts list
   - Click edit on a post
   - Make changes
   - Update categories and tags
   - Save draft or publish
